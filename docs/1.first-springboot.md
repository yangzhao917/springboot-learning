# Spring Boot是什么

SpringBoot是Java生态的一个用于构建Web应用的框架，提供了快速使用Spring的方式，在Spring的基础上通过约定优于配置的思想优化了Spring的配置方式，让开发人员有更多的时间可以专注于业务中，从而提高了开发效率。

Spring 的两个问题：
* 配置问题：
  Spring在整合其他框架时需要通过XML来配置相关内容，但XML配置并不好记，SpringBoot通过自动配置的方式在运行过程中来决定到底使用哪个配置。
* 依赖问题：
其次是和其他技术组合时的 jar 包依赖兼容性问题，这会导致浪费大量的时间去验证方案的可行性。SpringBoot通过起步依赖的方式来解决这个问题。

# 玩起来

## 项目结构

```
springboot-sample/
├── docs/
│   └── 1.first-springboot.md
├── src/
│   └── main/
│       └── java/
│           └── org/
│               └── example/
│                   ├── controller/
│                   │   └── SayController.java
│                   └── App.java
├── READMD.md
└── pom.xml

```

1. 主要组件说明
- Maven配置文件 (pom.xml)
  - 包含Spring Boot项目的基本配置
  - 定义了spring-boot-starter-web依赖 
2. 源代码目录 (src/main/java)
  - 主应用类: App.java
    - 包含@SpringBootApplication注解
    - 作为Spring Boot应用的入口点
  - 控制器类: SayController.java
    - 位于org.example.controller包中
    - 包含一个返回"Hello World!"的根路径映射
3. 文档文件
  - docs/1.first-springboot.md: Spring Boot介绍文档
  - READMD.md: 项目说明文档

## 1. pom文件

```
<!--    继承父 pom-->
<parent>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-parent</artifactId>
    <version>2.1.8.RELEASE</version>
</parent>

<!-- Web模块的起步依赖-->
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-web</artifactId>
</dependency>
```

## 2. 编写Controller

```java
@Controller
public class SayController {

    @ResponseBody
    @GetMapping("/")
    public String sayHello() {
        return "Hello World!";
    }
}
```

## 3. 编写启动类

```java
/**
 * 引导类，启动 Spring Boot项目的入口
 */
@SpringBootApplication
public class App {
    public static void main(String[] args) {
        System.out.printf("Hello and welcome!");

        // 启动方法
        SpringApplication.run(App.class, args);
    }
}
```

# 起步依赖

1. spring-boot-starter-parent

查看 pom.xml 文件 可以看到，spring-boot-starter-parent 项目继承了 spring-boot-dependencies 项目。

![springboot-dependencies.png](img/springboot-dependencies.png)

在spring-boot-dependencies 项目中通过 dependencyManagement 定义了 Spring Boot 所依赖的版本信息，比如 Spring Boot 的版本、Spring Framework 的版本、Spring Data 的版本等等。

![depm.png](img/depm.png)

2. spring-boot-starter-web
在 spring-boot-starter-web 中定义了 Spring Boot Web 应用所依赖的 jar 包。
